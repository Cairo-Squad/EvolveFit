🏗️ Architecture

This project is based on a Kotlin Multiplatform (KMP) architecture, enabling shared business logic between Android and iOS while maintaining native UIs on each platform.

🔧 Technologies

[!TIP]

Kotlin Multiplatform (KMP)

Jetpack Compose Multiplatform (CMP)

Room Database

Koin (Dependency Injection)

Coil (Image Loading)

Ktor (Networking)

Navigation 2 (Type-Safe)
